# CrediShoping — Sistema de Ventas a Crédito

## Descripción del Proyecto
Aplicación fullstack para gestión de ventas a crédito. Permite registrar clientes, productos, ventas en cuotas, pagos, compras de inventario y auditoría de operaciones.

---

## Estructura del Proyecto

```
crediShoping/
├── client/          # Frontend React 19 + Vite 7
│   └── src/
│       ├── views/       # Páginas (una por módulo)
│       ├── components/  # Formularios reutilizables
│       ├── App.jsx      # Router principal
│       └── App.css      # Variables CSS + overrides de Bootstrap
└── server/          # Backend Express 5 + Node.js
    ├── server.js        # Entry point
    ├── routes/          # Rutas Express (un archivo por módulo)
    ├── src/
    │   ├── controllers/ # Lógica HTTP (request/response)
    │   └── models/      # Consultas SQL a la base de datos
    └── db/
        ├── config.js    # Pool de conexión PostgreSQL
        └── schema/
            ├── DDL.sql  # Estructura de tablas, triggers e índices
            └── DML.sql  # Datos iniciales
```

---

## Stack Tecnológico

### Backend
- **Runtime**: Node.js con `"type": "module"` (ES Modules — usar `import/export`, nunca `require`)
- **Framework**: Express 5.x
- **Base de datos**: PostgreSQL 16 via librería `pg` (connection pool)
- **Auth**: `jsonwebtoken` + `bcryptjs` (pendiente de implementar)
- **Variables de entorno**: `dotenv` (.env en `/server/.env`)
- **Dev**: `nodemon`

### Frontend
- **Framework**: React 19 con Vite 7
- **Routing**: React Router DOM 7
- **UI**: Bootstrap 5.3 + Bootstrap Icons
- **HTTP Client**: Axios
- **Estilos**: CSS custom properties en `App.css` + clases Bootstrap

### Base de Datos
- **Motor**: PostgreSQL
- **Nombre de DB**: `credishoping`
- **Tablas**: `customers`, `products`, `admins`, `sales`, `sale_details`, `payments`, `shopping`, `audit_logs`
- **Triggers**: `increase_stock` (INSERT en shopping), `decrease_stock` (BEFORE INSERT en sale_details)

## Convenciones de Código

### Backend

**Rutas** (`routes/`):
```js
import { Router } from "express";
import { getX, createX } from "../src/controllers/xController.js";

const router = Router();
router.get("/x", getX);
router.post("/x", createX);
export default router;
```

**Controladores** (`src/controllers/`):
```js
import * as xModel from "../models/xModel.js";

// GET /x
export const getX = async (req, res) => {
  try {
    const data = await xModel.getAllX();
    res.json(data);
  } catch (error) {
    console.error(error);
    res.status(500).json({ error: "Error al obtener X" });
  }
};

// POST /x
export const createX = async (req, res) => {
  try {
    const { field1, field2 } = req.body;
    if (!field1) return res.status(400).json({ error: "field1 es obligatorio" });
    const result = await xModel.createX(req.body);
    res.status(201).json(result);
  } catch (error) {
    console.error(error);
    if (error.code === "23505") return res.status(400).json({ error: "Ya existe" });
    res.status(500).json({ error: "Error al crear X" });
  }
};
```

**Modelos** (`src/models/`):
```js
import pool from "../../db/config.js";

export const getAllX = async () => {
  const result = await pool.query("SELECT * FROM x ORDER BY id");
  return result.rows;
};

export const createX = async (data) => {
  const { field1, field2 } = data;
  const result = await pool.query(
    `INSERT INTO x (field1, field2) VALUES ($1, $2) RETURNING *`,
    [field1, field2]
  );
  return result.rows[0];
};
```

**Transacciones** (para operaciones multi-tabla):
```js
const client = await pool.connect();
try {
  await client.query("BEGIN");
  // ... operaciones ...
  await client.query("COMMIT");
  return result;
} catch (error) {
  await client.query("ROLLBACK");
  throw error;
} finally {
  client.release();
}
```

**Registro de nueva ruta en server.js**:
Importar y registrar con `app.use(xRoutes)` luego de los middlewares.

### Frontend

**Vistas** (`src/views/`):
```jsx
import { useState, useEffect } from 'react';
import axios from 'axios';
import FormX from '../components/FormX';

const X = () => {
  const [items, setItems] = useState([]);
  const [showForm, setShowForm] = useState(false);

  useEffect(() => {
    axios.get('http://localhost:3000/x')
      .then(res => setItems(res.data))
      .catch(err => console.error(err));
  }, []);

  return (
    <>
      <div className='d-flex justify-content-between'>
        <h5>X</h5>
        <button className='btn btn-primary' onClick={() => setShowForm(true)}>
          + Nuevo X
        </button>
      </div>
      {showForm && <FormX onClose={() => setShowForm(false)} />}
      <div className="rounded shadow overflow-hidden">
        <table className="w-100">
          <thead className="dash">
            <tr>
              <th className="px-4 py-2">Campo</th>
              <th className="px-4 py-2">Acciones</th>
            </tr>
          </thead>
          <tbody>
            {items.length === 0 ? (
              <tr><td className='text-center px-4 py-5 text-secondary' colSpan={2}>No hay registros</td></tr>
            ) : items.map(item => (
              <tr key={item.id}>
                <td className="px-4 py-2">{item.field}</td>
              </tr>
            ))}
          </tbody>
        </table>
      </div>
    </>
  );
};

export default X;
```

**Componentes de formulario** (`src/components/`):
```jsx
import { useState } from 'react';
import axios from 'axios';

const FormX = ({ onClose, onSaved }) => {
  const [form, setForm] = useState({ field1: '', field2: '' });
  const [error, setError] = useState('');

  const handleChange = (e) => setForm({ ...form, [e.target.name]: e.target.value });

  const handleSubmit = async (e) => {
    e.preventDefault();
    try {
      const res = await axios.post('http://localhost:3000/x', form);
      onSaved?.(res.data);
      onClose?.();
    } catch (err) {
      setError(err.response?.data?.error || 'Error al guardar');
    }
  };

  return (
    <div className='border border-secondary-subtle rounded p-4'>
      <div className='d-flex justify-content-between'>
        <h3>Nuevo X</h3>
        <button type="button" className="btn-close" onClick={onClose} aria-label="Close"></button>
      </div>
      {error && <div className="alert alert-danger">{error}</div>}
      <form className="row g-3" onSubmit={handleSubmit}>
        <div className="col-md-12 text-start">
          <label htmlFor="field1" className="form-label">Campo 1</label>
          <input
            type="text"
            className="form-control"
            id="field1"
            name="field1"
            value={form.field1}
            onChange={handleChange}
          />
        </div>
        <div className="col-md-12 text-start">
          <button type="submit" className="btn btn-success">Guardar</button>
          <button type="button" className="btn btn-danger ms-3" onClick={onClose}>Cancelar</button>
        </div>
      </form>
    </div>
  );
};

export default FormX;
```

## Reglas de Estilo Importantes

- **ES Modules siempre**: `import/export`, nunca `require/module.exports`
- **Arrow functions** para componentes y handlers
- **Nombres de archivos**: PascalCase para componentes React, camelCase para todo lo demás
- **Importaciones**: usar rutas con extensión `.js` en el backend
- **JSX**: usar `className` (no `class`), `htmlFor` (no `for`)
- **Botón cerrar formulario**: pasar `onClose` como prop, nunca inline `onClick={FormX}`
- **Mensajes de error del servidor**: en español, descriptivos
- **SQL**: parámetros posicionales `$1, $2...`, siempre `RETURNING *` en INSERT/UPDATE
- **CSS**: usar las variables CSS definidas en `App.css` (`var(--primary-color)`, etc.)
